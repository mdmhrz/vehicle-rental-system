Vehicle Rental System API

Welcome! This is the backend API for managing vehicles, bookings, users, and authentication.

View API Documentation

Available Endpoints:

  • /api/v1/auth - User authentication
  • /api/v1/users - Manage users
  • /api/v1/vehicles - Manage vehicles
  • /api/v1/bookings - Manage bookings

For any questions, contact the backend admin.
